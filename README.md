# Vanish-sales-analysis-Mexico

## Overview
This project analyzes the sales performance of Vanish products in Mexico using real (anonymized) business data.

## Objective
Identify key sales drivers, trends, and actionable business insights.

## Tools
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook

## Data
The dataset includes:
- Sales transactions (FACT table)
- Product, category, segment, and calendar dimensions

## Key Analysis
- Data cleaning and preprocessing
- Data merging and modeling
- Exploratory data analysis (EDA)
- Sales trends and patterns
- Business insights

## Results
The analysis identifies key variables impacting sales performance and highlights opportunities for business optimization.

## Note
All data is anonymized and used strictly for analytical purposes.
